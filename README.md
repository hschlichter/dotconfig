# .Config files

## Alacritty
<https://alacritty.org>

or

install via homebrew

```
$ brew install alacritty
```

## fzf
<https://github.com/junegunn/fzf>

```
$ brew install fzf

$(brew --prefix)/opt/fzf/install
```

## Neovim
```
$ brew install nvim
```

Run `:PackerUpdate` after first open.
